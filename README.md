# PHP CRUD Project

## Description

This is a simple PHP CRUD (Create, Read, Update, Delete) project that allows you to manage records in a database.

## Features

- Create new records
- Read and display records
- Update existing records
- Delete records

## Technologies Used

- PHP
- MySQL
- HTML/CSS
- Bootstrap

## Requirements

- PHP 7.0 or higher
- MySQL database server
- Web server (e.g., Apache)
